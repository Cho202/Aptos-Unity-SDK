\doxysubsection{Detailed Description}
The \mbox{\hyperlink{namespace_aptos}{Aptos}} REST Client contains a set of \mbox{[}standalone\mbox{]} Coroutines that can started within any Unity script. 

Consideration must be placed into the wait time required for a transaction to be committed into the blockchain.
